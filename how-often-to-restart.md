# How often to restart my Linux Computer? </br>  
As a Windows System Administrator, I have always restarted computers before and after a major software installation or applying patches. </br>  
Now that I am working with Linux and updating the OS frequently, I wanted to know how often should I restart my computer. </br>  
By reading a few blog posts, I learned that I can run commands after applying software updates to find if a reboot is required. </br>  
The command **sudo dnf needs-restarting -r** can provide information whether a restart is required. </br>
